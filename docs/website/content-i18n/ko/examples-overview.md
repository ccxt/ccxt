### JavaScript

[JavaScript 예제](/docs/examples/js/)

```bash
node path/to/example.js # substitute for actual filename here
```

이 예제들은 [xpl](https://github.com/xpl)의 다음과 같은 매우 유용한 고품질 Node.js 모듈이 필요할 수 있습니다:

- [ololog](https://github.com/xpl/ololog): 플랫폼 독립적 로깅 with 블랙잭과 후커 ([ololog @ npm](https://npmjs.com/package/ololog))
- [ansicolor](https://github.com/xpl/ansicolor): ANSI 색상/스타일 관리를 위한 고품질 JavaScript 라이브러리 ([ansicolor @ npm](https://npmjs.com/package/ansicolor))
- [as-table](https://github.com/xpl/as-table): 객체를 ASCII 테이블로 출력하는 간단한 함수 ([as-table @ npm](https://npmjs.com/package/as-table))

위의 모든 모듈은 npm에 의해 ccxt 라이브러리 devDependencies와 함께 자동으로 설치됩니다.
