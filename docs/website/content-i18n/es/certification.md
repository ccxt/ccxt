# Programa de Certificación CCXT · ![CCXT Certified](https://img.shields.io/badge/CCXT-certified-green.svg)

La estructura de CCXT define un estándar bueno, portable y compatible entre plataformas para las interfaces de API de los exchanges, que está implementado en la API Unificada de CCXT. Los exchanges son bienvenidos a solicitar nuestro programa de certificación. La certificación está supervisada técnicamente y con garantía de calidad por miembros del Equipo de Desarrollo de CCXT. Esto implica que un exchange que posea el distintivo de "certificado" está correctamente implementado y probado por los autores de CCXT. La certificación significa menos errores, más funcionalidad, soporte prioritario y una implementación en general mucho más estable y eficiente.

## Requisitos

Para integrarse y certificarse, el exchange debe implementar una API de calidad. Por favor, consulte la lista completa de requisitos técnicos aquí: /docs/requirements
