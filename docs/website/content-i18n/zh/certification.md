# CCXT 认证计划 · ![CCXT Certified](https://img.shields.io/badge/CCXT-certified-green.svg)

CCXT 的结构为交易所的 API 接口定义了一套良好、可移植且跨兼容的标准，该标准已在 CCXT 统一 API 中实现。欢迎交易所申请我们的认证计划。认证工作由 CCXT 开发团队成员进行技术监督和质量保证。这意味着获得"已认证"徽章的交易所已由 CCXT 的作者进行了正确的实现和测试。认证意味着更少的错误、更多的功能、优先支持以及总体上更稳定、更高效的实现。

## 要求

接入并获得认证要求交易所实现高质量的 API。请在此处查看完整的技术要求列表：/docs/requirements
