# Programa de Certificação CCXT · ![CCXT Certified](https://img.shields.io/badge/CCXT-certified-green.svg)

A estrutura do CCXT define um padrão bom, portátil e compatível entre plataformas para interfaces de API de exchanges, que é implementado na API Unificada do CCXT. As exchanges são bem-vindas a candidatar-se ao nosso programa de certificação. A certificação é supervisionada tecnicamente e tem a qualidade assegurada por membros da Equipa de Desenvolvimento do CCXT. Isso implica que uma exchange com o selo "certificada" está devidamente implementada e testada pelos autores do CCXT. A certificação significa menos bugs, mais funcionalidade, suporte prioritário e uma implementação muito mais estável e eficiente em geral.

## Requisitos

Para ser integrada e certificada, a exchange deve implementar uma API de qualidade. Por favor, consulte a lista completa de requisitos técnicos aqui: /docs/requirements
