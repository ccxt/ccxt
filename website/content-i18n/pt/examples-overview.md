### JavaScript

[Exemplos de JavaScript](/docs/examples/js/)

```bash
node path/to/example.js # substituir pelo nome do arquivo real aqui
```

Estes exemplos podem exigir os seguintes módulos Node.js super úteis de alta qualidade por [xpl](https://github.com/xpl):

- [ololog](https://github.com/xpl/ololog): Registro multiplataforma com blackjack e prostitutas ([ololog @ npm](https://npmjs.com/package/ololog))
- [ansicolor](https://github.com/xpl/ansicolor): Uma biblioteca JavaScript de qualidade para gerenciamento de cor/estilo ANSI ([ansicolor @ npm](https://npmjs.com/package/ansicolor))
- [as-table](https://github.com/xpl/as-table): Uma função simples que imprime objetos como tabelas ASCII ([as-table @ npm](https://npmjs.com/package/as-table))

Todos os módulos acima são instalados automaticamente pelo npm com as dependências de desenvolvimento da biblioteca ccxt.
