## 贡献指南

请确保您的 pull request 符合以下要求：

- 在提交新建议前，先搜索已有建议，确保不重复。
- 每项建议单独提交一个 pull request。
- 使用以下格式：[名称](链接) - 描述。
- 保持描述简洁。
- 欢迎新增分类或改进现有分类。
- 所有描述均以句号结尾。
- 尽量确保您的 pull request 只有一个提交，提交前可将多个提交压缩为一个。
- 尽量详细填写 pull request 的标题和描述，说明为什么某个资源值得加入列表。
- 确保新增内容按字母顺序排列，包括标题和条目本身。
- 确保您提交的资源质量高、文档完善且近期有维护。若其受欢迎程度较高，也有助于我在将其加入列表前进行研究。
- 本列表不用于宣传个人资源。在质量和功能相近的情况下，免费或开源资源优先于付费资源。
